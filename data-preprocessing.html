Data Cleaning

Data cleaning is like tidying up your room - we need to handle missing items (values), remove unusual items (outliers), and fix inconsistencies in our data.

Missing Values

Imagine a survey where some people skipped questions. We can:

  • Remove incomplete responses (drop rows)
  • Fill with average values (mean imputation)
  • Use most common values (mode imputation)

Python Example:


import pandas as pd
import numpy as np

# Load sample dataset
df = pd.read_csv('customer_data.csv')

# Check missing values
print("Missing values:\n", df.isnull().sum())

# Method 1: Drop rows with missing values
df_cleaned = df.dropna()

# Method 2: Fill missing values with mean
df['age'] = df['age'].fillna(df['age'].mean())

# Method 3: Fill missing values with mode
df['category'] = df['category'].fillna(df['category'].mode()[0])

Different approaches to handling missing values

Outliers

Outliers are unusual values that don't follow the normal pattern. Like finding a 150-year-old person in age data.

Python Example:


# Detect outliers using IQR method
Q1 = df['age'].quantile(0.25)
Q3 = df['age'].quantile(0.75)
IQR = Q3 - Q1

# Define outlier boundaries
lower_bound = Q1 - 1.5 * IQR
upper_bound = Q3 + 1.5 * IQR

# Remove outliers
df_no_outliers = df[(df['age'] >= lower_bound) & 
                    (df['age'] <= upper_bound)]

Inconsistencies

Fix data that doesn't match expected formats or values.

Python Example:


# Fix inconsistent categories
df['category'] = df['category'].str.lower()
df['category'] = df['category'].replace({
    'cat1': 'category_1',
    'cat2': 'category_2'
})

# Fix date format inconsistencies
df['date'] = pd.to_datetime(df['date'])
