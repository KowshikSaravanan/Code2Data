Machine Learning

Machine Learning is a subset of AI that enables systems to learn patterns from data and make decisions with minimal human intervention. It includes various paradigms such as supervised, unsupervised, deep, and reinforcement learning.

Data Preprocessing - Prepares raw data for modeling.

  • Data Cleaning - Handle missing values, outliers, and inconsistencies.
  • Feature Scaling - Normalize features (StandardScaler, MinMax).
  • Feature Engineering - Create new features from raw data.
  • Handling Categorical Variables - One-hot and label encoding.
  • Transformation - Log, Box-Cox â€“ reduce skewness.
  • Dimensionality Reduction - PCA, t-SNE, UMAP.
  • Data Sampling - Address imbalanced classes via under/oversampling.

Supervised Learning

Learning from labeled data (input features with known output labels).

Regression

Predicts continuous output variables.

  • Linear Regression
  • Polynomial Regression
  • Ridge Regression
  • Lasso Regression
  • ElasticNet Regression
  • Support Vector Regression (SVR)
  • Decision Tree Regression
  • Random Forest Regression
  • Gradient Boosting Regression

Classification

Predicts discrete labels (e.g., Yes/No, Category A/B/C).

  • Logistic Regression
  • Support Vector Machine (SVM)
  • Decision Trees
  • Naive Bayes
  • k-Nearest Neighbors (kNN)
  • Random Forest
  • Gradient Boosting Machines (GBM)
  • Neural Networks
  • Linear Discriminant Analysis (LDA)
  • Quadratic Discriminant Analysis (QDA)

Unsupervised Learning

Discovers hidden patterns from unlabeled data.

Clustering

  • K-Means
  • Hierarchical Clustering
  • DBSCAN
  • Gaussian Mixture Models (GMM)
  • Self Organizing Maps (SOMs)

Dimensionality Reduction

  • PCA (Principal Component Analysis)
  • t-SNE
  • LDA (Linear Discriminant Analysis)
  • Autoencoders

Deep Learning

Subset of ML using deep neural networks with many layers.

  • Artificial Neural Networks (ANNs)
  • Convolutional Neural Networks (CNNs)
  • Recurrent Neural Networks (RNNs)
  • Long Short-Term Memory (LSTM)
  • Gated Recurrent Units (GRUs)
  • GANs (Generative Adversarial Networks)
  • Transformers
  • Self-Attention

Reinforcement Learning

Training agents through rewards and penalties in an environment.

  • Q-Learning
  • Deep Q-Networks (DQNs)
  • SARSA
  • Actor-Critic Methods
  • Policy Gradients
  • Temporal Difference Learning

Model Evaluation and Selection

  • Cross Validation
  • Hyperparameter Tuning
  • Regularization
  • Metrics
  • Model Selection Criteria

Specific Applications

  • Natural Language Processing (NLP)
  • Computer Vision
  • Time Series Analysis
  • Recommendation Systems
  • Robotics
